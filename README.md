# GEC Computer Engineering

Compilers used:

1. gcc ver 9.1.0
2. g++ ver 9.1.0

- How to compile c code using gcc: <br> 
`gcc fileName.c -o outputFileName`

- How to compile c++ code using g++: <br> 
`g++ fileName.c -o outputFileName`

Add `-lm` flag for linking math.h

How to run code:
`./outputFileName`
