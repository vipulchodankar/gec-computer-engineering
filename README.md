# GEC Computer Engineering

- Compiling and running c code using gcc: <br> 
`gcc fileName.c -o outputFileName`<br> 
`./outputFileName`

- Compiling and running c++ code using g++: <br> 
`g++ fileName.c -o outputFileName`<br> 
`./outputFileName`

Add `-lm` flag for linking math.h

<hr>

- Compiling and running java code using javac: <br> 
`javac ClassName.java`<br>
`java ClassName`

- Running python: <br> 
`python fileName.py`<br>
